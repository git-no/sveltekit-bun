# SvelteKit with Bun

SvelteKit using Bun.
Works with Vercel or Docker.

Starter template with modules/solutions:

- SvelteKit
- TailwindCSS
- Bun
- Typescript
- ESLint
- Prettier
- Svelte MCP
- Zod

Includes:

- SiteMap

## Development

```sh
bun --bun run dev
```

## Building

To create a production version of your app:

```sh
bun --bun run build
```
