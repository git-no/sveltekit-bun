# Project general coding standards

## About this project and repository

This repository contains the code for the project. It includes various components, utilities, and configurations that are essential for the development and maintenance of the project. The code is organized in a way that promotes modularity, readability, and maintainability.

It uses:

- typescript version 6
- svelte version 5 and sveltekit
- tailwindCSS 4
- DaisyUI
- eslint

For mcp servers of these packages follow the links in the file `./mcp-config.json`.

## Naming Conventions

- Use PascalCase for component names, interfaces, and type aliases
- Use camelCase for variables, functions, and methods
- Prefix private class members with underscore (\_)
- Use ALL_CAPS for constants
